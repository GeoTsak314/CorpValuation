# CORP Valuation app v4.1 by G. Tsakalos

Desktop εφαρμογή σε **Python / Tkinter / SQLite** για αποτίμηση επιχειρήσεων, εισαγωγή οικονομικών δεδομένων, υπολογισμό χρηματοοικονομικών δεικτών, σύγκριση εταιρειών και εξαγωγή αναφορών.

## Βασικά χαρακτηριστικά

- Διαχείριση εταιρειών
- Καταχώριση **Ισολογισμού** ανά εταιρεία και έτος
- Καταχώριση **Αποτελεσμάτων Χρήσης** ανά εταιρεία και έτος
- **Import από Excel template**
- Αυτόματος υπολογισμός χρηματοοικονομικών δεικτών
- Tab **Δείκτες / Report** με σημειώσεις ανά δείκτη
- Tab **Compare** για σύγκριση όλων των εταιρειών για συγκεκριμένο έτος
- Export αναφορών σε **PDF** και **XLSX**
- Export comparison board σε **XLSX**
- Τοπική αποθήκευση σε **SQLite**
- Ρυθμίσεις μέσω αρχείου **app.cfg**
- Άνοιγμα URL πεδίων από το Companies tab
- Mouse wheel scrolling στα scrollable σημεία της εφαρμογής

## Αρχεία έργου

- `app.py`: ο βασικός κώδικας της εφαρμογής
- `app.cfg`: αρχείο ρυθμίσεων
- `corp_values.sqlite`: προεπιλεγμένη βάση SQLite
- `requirements.txt`: απαιτούμενες βιβλιοθήκες
- `IndexMap.jpg`: διάγραμμα index map που ανοίγει από το menu

## Tabs εφαρμογής

### 1. Companies
Καταχώριση και διαχείριση βασικών στοιχείων εταιρείας:

- Επωνυμία
- ΑΦΜ
- ΓΕΜΗ Αρ.
- ΓΕΜΗ URL
- URL εταιρείας
- Σημειώσεις

Διαθέσιμες ενέργειες:
- `New`
- `Save`
- `Edit selected`
- `Search`
- `Import Excel`
- `Open` για τα URL πεδία

Η αναζήτηση γίνεται σε:
- Επωνυμία
- ΑΦΜ
- ΓΕΜΗ

### 2. Ισολογισμός
Καταχώριση κονδυλίων ισολογισμού για συγκεκριμένη εταιρεία και έτος.

Περιλαμβάνει:
- βασικά πεδία ενεργητικού / παθητικού
- πεδία `Διαφορά` ανά βασική ενότητα
- αυτόματα υπολογιζόμενα totals
- πεδίο `Σχόλια`

### 3. Αποτελέσματα Χρήσης
Καταχώριση οικονομικών αποτελεσμάτων για συγκεκριμένη εταιρεία και έτος.

Περιλαμβάνει:
- βασικά πεδία εσόδων / εξόδων
- πεδίο `Άλλα Έξοδα`
- χειροκίνητο πεδίο `EBITDA`
- αυτόματα υπολογιζόμενα totals όπως EBIT / PBT / PAT
- πεδίο `Σχόλια`

### 4. Δείκτες / Report
Υπολογισμός και εμφάνιση χρηματοοικονομικών δεικτών για την επιλεγμένη εταιρεία.

Υποστηρίζει:
- επιλογή εταιρείας
- υπολογισμό δεικτών για όλα τα διαθέσιμα έτη
- σημειώσεις **ανά δείκτη**
- export σε **PDF**
- export σε **XLSX**

Το PDF report περιλαμβάνει:
- τίτλο με την επωνυμία της εταιρείας
- πίνακα δεικτών ανά έτος
- σημειώσεις ανά δείκτη
- footer: `Corp Valuation app - by G.Tsakalos`
- επιπλέον σελίδα με 3 charts:
  - EBIT
  - ROA
  - ROE

Το XLSX report περιλαμβάνει:
- sheet `Ratios`
- sheet `Totals`
- sheet `Charts`

### 5. Compare
Σύγκριση όλων των εταιρειών για συγκεκριμένο έτος.

Περιλαμβάνει:
- dropdown επιλογής έτους
- κουμπί `Υπολογισμός`
- board με:
  - γραμμές = εταιρείες
  - στήλες = δείκτες
- sorting πατώντας στις επικεφαλίδες των στηλών
- export της σύγκρισης σε **XLSX**

## Import από Excel

Η εφαρμογή διαβάζει Excel αρχεία που περιέχουν τα sheets:

- `Κατάσταση Οικονομικής Θέσης`
- `Κατάσταση Συνολικού Εισοδήματος`

Κατά το import:
- διαβάζει τα έτη από τις επικεφαλίδες του Excel
- γεμίζει αυτόματα balance sheet και income sheet ανά έτος
- συνδέει τα δεδομένα με υπάρχουσα εταιρεία ή δημιουργεί νέα

## Αρχείο ρυθμίσεων `app.cfg`

Η εφαρμογή δημιουργεί / χρησιμοποιεί αρχείο `app.cfg` στον ίδιο φάκελο.

Παράδειγμα:

```ini
[app]
db_path = corp_values.sqlite
index_map_path = IndexMap.jpg
```

Ρυθμίσεις:
- `db_path`: path της SQLite βάσης που ανοίγει αυτόματα στο startup
- `index_map_path`: path του διαγράμματος Index Map που ανοίγει από το menu

## Menu εφαρμογής

Το menu περιλαμβάνει:

- `OpenDB`: άνοιγμα άλλης SQLite βάσης
- `SaveDB`: αποθήκευση της τρέχουσας βάσης
- `ConfigFileEdit`: άνοιγμα του `app.cfg` με τον default editor
- `IndexMap`: άνοιγμα του αρχείου index map

## Εγκατάσταση

Σε Windows:

```bat
python -m pip install -r requirements.txt
python app41.py
```

## Δημιουργία .exe

Build σε Windows executable με PyInstaller:

```bat
pyinstaller --onefile --windowed app41.py
```

## Σημαντικές σημειώσεις

- Για ορισμένους δείκτες χρειάζεται και το προηγούμενο έτος, άρα υπολογίζονται πληρέστερα όταν υπάρχουν δεδομένα για τουλάχιστον 2 συνεχόμενα έτη.
- Το `tkinter` είναι μέρος της standard Python εγκατάστασης στα Windows και δεν χρειάζεται συνήθως ξεχωριστό install.
- Η εφαρμογή χρησιμοποιεί Unicode-compatible fonts για σωστή εμφάνιση ελληνικών στα PDF, όταν υπάρχει διαθέσιμη κατάλληλη γραμματοσειρά στο σύστημα.
